alt + enter 해당 오류 위에 놓고 사용시 수정 가능 오류 없는 코드에서도 더 길거나 짧은 코드로 수정 가능
F2 다음 오류, 경고,제안으로 이동
cmd + 1 (Mac) Alt + 1 (win) 프로젝트 창을 열기
Esc 에디터로 돌아옴
cmd + E (Mac) Ctrl + E (Win) 가장 최근 열었던 파일 확인하기
cmd + B (Mac) ctrl + B (Win) 심볼의 선언으로 이동하기
Alt + F7 모든 사용위치 표기
ctrl 더블클릭 모든 실행목록이 열림
alt + 위아래 방향키 (Mac) ctrl+W , ctrl+Shift+W (Win) 커서 주변의 코드 섹션을 확장 또는 축소
cmd + / (Mac) ctrl + / (Win) 간편하게 주석 추가
shift + cmd + Enter (Mac) shift + ctrl + Enter (Win) 구문 완성
alt + cmd + L (Mac) ctrl + alt + L (Win) 파일 서식을 프로젝트 표준에 맞추기
ctrl + T (Mac) shift ctrl + alt + T (Win) 모든 리팩토링 표시
shift + cmd + A (Mac) shift + ctrl + A (Win) 단축키 검색
shift 더블 탭 전체 검색
